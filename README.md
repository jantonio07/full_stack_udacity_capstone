# full_stack_udacity_capstone

I would say the core of internet is sharing content with other: documents, images, videos, etc.

Because of that, the aim of this project is having a first approach to image handling in web pages.
Particularly, developing a site to share images in albums. Any who accesses the site will be able
to see the content, but only people with right permissions will be allowed to create albums and
upload images.
